## Setup

-   Setup environment

    https://reactnative.dev/docs/environment-setup

-   Install dependencies

```sh
yarn install
```

-   Install pods (make sure you are inside ios folder)

```sh
pod install
```

-   Launch application

```sh
yarn run android / ios
```
